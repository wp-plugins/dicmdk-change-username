== Description ==

Changing usernames in Wordpress is currently annoying, and without external software it is really a pain. With this plugin you can enter the username you want to change and enter the new username. Then submit.

The username will now be changed and you can use it to login with.

Created to help people change from "Admin" username to something else, without having to use phpmyadmin or similer tools. 

Does not work on multisites (ask for update if needed).


== Installation ==

1. Upload `dicmdk-changeusername.php`to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. If you do not have dicm.dk - Toolbox, install it. (https://wordpress.org/plugins/dicmdk-toolbox/)
